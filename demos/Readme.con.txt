This is a small demo application that
demonstrates how to capture output from
console applications into a GUI application.
It works great as far as I know.
I adapted it for Kol from the original source
by:

  Martin Lafferty
  martinl@prel.co.uk

  Production Robots Engineering Ltd
  Box 2290, Wimborne, Dorset, BH21 2YY, England.

That took just two minutes, so beware: No Warranty and
no Guarantee that it will always work.
I use it in my own KOL Design Environment (Kolide)
to spawn DCC32 etc.

The example can acually act as a compiler front end:
Just check the path to your delphi DCC32.exe and
fill in a project name at the Command options.
